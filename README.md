# Digital Card Builder

Create and share your digital business card instantly. Choose from multiple premium card templates, customize colors, and share via link or QR code.

## Features

- Multiple card design templates
- Custom accent colors
- vCard export / contact download
- Shareable card links
- QR code generation
- Image export

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
